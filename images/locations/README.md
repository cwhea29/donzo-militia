# Location Images

Place all your location/marker popup images in this folder.

## Usage
When adding a marker via the map, enter just the **filename** in the "Image Filename" field.

**Example:**
- File placed here: `images/locations/maze_bank.jpg`
- Enter in the field: `maze_bank.jpg`

## Tips
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
- Recommended size: 600×400px or similar widescreen ratio (displayed at 300×170px in popup)
- Use descriptive filenames — e.g. `cayo_compound_gate.jpg` not `img001.jpg`
- Keep file sizes under 500KB for fast loading
