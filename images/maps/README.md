# Map Images

Place your GTA V map images in this folder with the following exact filenames:

## Los Santos
| Filename                    | Description              |
|-----------------------------|--------------------------|
| `GTAV_ATLUS.jpg`            | Atlas / overview map     |
| `GTAV-HD-MAP-roadmap.jpg`   | High-def roadmap         |
| `GTAV-HD-MAP-satellite.jpg` | Satellite view           |

## Cayo Perico
| Filename                    | Description              |
|-----------------------------|--------------------------|
| `GTAV-CAYO-atlas.jpg`       | Cayo Perico atlas        |
| `GTAV-CAYO-roadmap.jpg`     | Cayo Perico roadmap      |
| `GTAV-CAYO-satellite.jpg`   | Cayo Perico satellite    |

## Notes
- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
- If you use different filenames, update the `MAP_IMAGES` object in `js/config.js`
- Higher resolution images look better when zoomed in, but load slower
- Recommended size: 4096Ã—4096px or higher for best zoom quality
